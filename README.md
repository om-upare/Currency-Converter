# Currency Converter 💵💱

A simple and lightweight Currency Converter built using HTML, CSS, and JavaScript. This tool allows users to convert amounts between different currencies with an easy-to-use interface.

## Features

- **User-friendly Interface:** Clean and simple design for effortless currency conversion.
- **Quick Conversion:** Input your amount, select the source and target currencies, and click on the Exchange Rate Button.
- **Responsive Design:** Works well on both desktop and mobile browsers.
